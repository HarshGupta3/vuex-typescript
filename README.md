# vue-next-webpack-preview

> Minimal webpack setup for Vue 3 (beta) 

### Refrence URL
-https://dev.to/3vilarthas/vuex-typescript-m4j

### Prerequisites
- Node & NPM

### Install
```sh
npm install
```
### Usage
##### Develop
```sh
# run dev server at localhost:8080
npm run dev
```
##### Build
```sh
# transpile js for deployment
npm run build
```
